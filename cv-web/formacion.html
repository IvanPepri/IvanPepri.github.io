Formación Académica

Universidad Politécnica de Madrid
Escuela Técnica Superior de Ingenieros en Topografía, Geodesia y Cartografía

Durante mi formación, he adquirido una sólida base en herramientas GIS [QGIS, ArcGIS Pro, ArcGIS Online] para el análisis espacial y resolución de problemas, potenciando este uso de herramientas mediante la programación. He desarrollado habilidades en lenguajes como [Python,JavaScript, HTML y CSS, SQL, R] para automatizar tareas, gestionar bases de datos espaciales y realizar análisis complejos más allá de las herramientas estándar de GIS.

Además, he aplicado estos conocimientos en áreas especificas como:

  • Ecosistema ESRI: Experiencia en el análisis de datos y simbología avanzada con ArcGIS Pro, y en el despliegue de soluciones web mediante ArcGIS Online (visores, DashBoards, StoryMaps y formularios de campo con Survey123).
  • Análisis Avanzado: Utilización de R para el análisis estadístico espacial y el modelado de series temporales.
  • Infraestructura de Datos Espaciales (IDE): Experiencia en la configuración y publicación de servicios web geoespaciales (WMS, WFS) mediante GeoServer y MapServer.
  • Business Intelligence (BI): Creación de dashboards e informes interactivos con Power BI, aplicando los fundamentos de la gestión de Big Data.
  • Desarrollo Web GIS: Experiencia en la creación de aplicaciones web geoespaciales, manejando el Frontend (HTML,CSS,JavaScript y librerías de mapas como Leaflet), y el Backend (Python con el framework Django). Capacidad para integrar datos consumiendo APIs REST (JSON/GeoJSON), estándares modernos como OGC APIs y servicios de infraestructuras oficiales (IDEE, INSPIRE).
